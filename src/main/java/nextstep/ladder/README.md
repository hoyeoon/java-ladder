# 기능 요구 사항

## 2단계 - 사다리(생성)

-[x] 참여할 사람 이름 입력 (,로 구분)
    - [x] 최대 다섯 글자
-[x] 사람(Person) 객체 구현
-[x] 사람들(People) 객체 구현
-[x] 사다리 높이 입력
-[x] 사다리 가로 라인 만들기(라인(Line) 객체 구현)
    -[x] 라인을 생성 할지/말지 판단하는 1/2 랜덤 함수 사용
    -[x] 가로 라인 확인 (겹치지 않기 위해)
-[x] 사다리(Ladder) 객체 구현
-[ ] 실행결과 출력
    - [x] 이름 출력
    - [x] 사다리 출력
